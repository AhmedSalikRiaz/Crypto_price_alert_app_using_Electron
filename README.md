# Electron Crypto Price Alert App

This is a cross platform shoppinglist desktop application built on electron

## Quick Start

### Installation

Install the dependencies

```sh
$ npm install
```

### Run the application
To run electron

```sh
$ npm start
```

### To Package & Build

For Windows

```sh
$ npm run package-win
```

For Mac

```sh
$ npm run package-mac
```

For Linux

```sh
$ npm run package-linux
```
